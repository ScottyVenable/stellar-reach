# Galactic Trader

The `main` branch is reserved for **stable releases only**. There is no
stable release yet.

Active development happens on the [`development`][dev] branch. Public
testing builds live on the [`alpha`][alpha] branch.

[dev]: https://github.com/ScottyVenable/galactic-trader/tree/development
[alpha]: https://github.com/ScottyVenable/galactic-trader/tree/alpha

## Where to go

- Source, issues, and pull requests: [`development`][dev]
- Test builds and pre-release downloads: [`alpha`][alpha]
- Full history before the branch reorganisation: tag
  [`pre-reorg-backup-main`](https://github.com/ScottyVenable/galactic-trader/tree/pre-reorg-backup-main)

## Roadmap to v1.0

The first commit on this `main` branch will be `v1.0.0`. Until then,
everything in this branch is intentionally minimal.

See [the roadmap on `development`](https://github.com/ScottyVenable/galactic-trader/blob/development/docs/ROADMAP.md)
for the path to a stable release.
